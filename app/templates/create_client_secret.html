Set Up Google Drive API Integration

Follow these steps to enable Google Drive API:

To enable real Google Drive integration, you need to create API credentials in the Google Cloud Console:

  1. Go to the Google Cloud Console
  2. Create a new project (or select an existing one)
  3. Enable the Google Drive API for your project
  4. Go to "Credentials" and create an OAuth 2.0 Client ID
  5. Configure the OAuth consent screen (include your email address as a test user)
  6. For "Authorized redirect URIs", add: {{ redirect_uri }}
  7. Download the JSON credentials file
  8. Rename it to "client_secret.json" and place it in your app's root directory
After setting up the Google Drive API:

Restart your Flask application to apply the changes.

Upload the client_secret.json file you downloaded from Google Cloud Console.
